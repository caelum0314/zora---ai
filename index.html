什么是 Zora AI 助手？

Zora 是一个命令行界面的 AI 助手程序，它将大语言模型（LLM）与系统命令执行能力相结合，让你能够在终端中与一个活泼的虚拟助手对话，并让她帮你执行 Shell 命令、管理文件等。

功能特点

  • 智能对话：基于 OpenAI API 兼容的模型，支持自定义模型和接口地址
  • 命令执行：AI 可以通过在消息末尾添加 command: 标签调用系统 Shell
  • 上下文管理：自动保存最近的对话历史，支持手动压缩或清空上下文
  • 工作目录保持：AI 执行的命令会在当前工作目录下运行，支持 cd 命令
  • 记忆机制：AI 拥有专属的 MEMORY.md 文件，可写入重要信息
  • 技能扩展：通过配置文件可自定义可调用的外部命令列表
  • 飞书集成：支持通过命令发送消息到飞书，以及从飞书接收命令并执行
  • Telegram集成：支持通过命令发送消息到Telegram，以及从Telegram接收命令并执行
  • 远程控制：通过飞书或Telegram远程控制电脑执行命令，实现类似OpenClaw的功能

安装步骤

1. 克隆仓库或下载源码：
git clone https://github.com/yourname/zora.git
cd zora
2. 安装依赖：
pip install -r requirements.txt
3. 配置API密钥：

编辑 config.json 文件，填写你的 API 密钥和其他配置信息：

{ "openai": { "api_key": "your_openai_api_key", "model": "gpt-3.5-turbo", "base_url": "https://api.openai.com/v1" }, "telegram": { "bot_token": "your_telegram_bot_token", "chat_id": "your_chat_id" }, "feishu": { "app_id": "your_feishu_app_id", "app_secret": "your_feishu_app_secret", "chat_id": "your_feishu_chat_id" }, "system_prompt": "你是Zora，一个活泼可爱的AI助手，住在用户的终端里，帮助用户执行命令、管理文件、回答问题。" }
4. 配置飞书和Telegram（可选）：

飞书配置：

  1. 登录飞书开发者平台：https://open.feishu.cn/
  2. 创建一个企业自建应用
  3. 在"凭证与基础信息"中获取App ID和App Secret
  4. 在"权限管理"中添加"im:message:send"和"im:message:read"权限
  5. 在"事件订阅"中配置消息接收事件
  6. 在config.json中填写app_id、app_secret和chat_id（群聊或用户的chat_id）

Telegram配置：

  1. 在Telegram中搜索"@BotFather"
  2. 发送"/newbot"命令创建一个新机器人
  3. 获取机器人的token
  4. 与机器人聊天，然后访问 https://api.telegram.org/bot<token>/getUpdates 获取chat_id
  5. 在config.json中填写bot_token和chat_id

使用方法

在项目目录下运行主程序：

python main.py

内置命令

命令 作用
summary 压缩当前上下文，保留重要信息
clear 清空所有上下文历史
command <命令> 手动执行Shell命令
feishu <消息> 发送消息到飞书
telegram <消息> 发送消息到Telegram
exit 退出程序

远程控制功能

Zora支持通过飞书和Telegram远程控制电脑执行命令，实现类似OpenClaw的功能。

使用方法：

  1. 确保飞书或Telegram已正确配置（参考安装步骤4）
  2. 运行Zora主程序：python main.py
  3. 在飞书或Telegram中发送命令：

支持的远程命令格式：

  • 执行系统命令： command: <命令>，例如 command: ls -la
  • AI对话： ai: <问题>，例如 ai: 什么是机器学习？

常见问题

Q: 为什么命令执行失败？
A: 请确保命令格式正确，并且当前用户有执行该命令的权限。
Q: 为什么飞书或Telegram消息发送失败？
A: 请检查配置文件中的API密钥和相关参数是否正确。
Q: 如何修改AI助手的名字？
A: 编辑config.json文件中的system_prompt字段，将名字改为你喜欢的名称。
Q: 为什么远程控制功能不工作？
A: 请检查以下几点：
1. 飞书或Telegram的API密钥和参数是否正确配置
2. 网络连接是否正常
3. 飞书或Telegram的权限是否正确设置
4. Zora程序是否正在运行
Q: 远程控制支持哪些命令？
A: 远程控制支持执行系统命令和AI对话：
1. 执行系统命令：发送 command: <命令>
2. AI对话：发送 ai: <问题>
